Annotation

This bachelor thesis focuses on the simulation and synthetic data generation of object packing for computer vision and robotic learning applications. The primary objective is to develop a system capable of simulating the packing of 3D object models into various containers, such as boxes or sacks, while realistically modeling object interactions, deformations, and physical constraints. The system computes the packing simulation, generates corresponding surface textures and material properties for the containers, and renders high-quality synthetic images for use in neural network training. The generated dataset aims to support the development of vision-based algorithms for tasks such as object detection, segmentation, and pose estimation in cluttered or packed environments. This research contributes to the creation of scalable, controllable, and physically realistic synthetic datasets for advanced computer vision and robotic perception systems.

Implementation focuses on deformable bags / flexible packaging in UE5 (Chaos Cloth).

Goal

  • Design and implement a simulation framework for 3D object packing into containers such as boxes or sacks.
  • Model physical interactions and constraints during the packing process, including object placement, deformation, and collision handling.
  • Generate realistic surface textures and material properties for containers and packed scenes.
  • Render high-quality synthetic images and corresponding ground truth annotations suitable for training computer vision models.
  • Evaluate the generated synthetic data in the context of training neural networks for object segmentation and pose estimation tasks.

Assignment

  • UE5 scene and asset pipeline (bags + rigid objects)
  • Deformable bag simulation (Chaos Cloth) with stable collisions
  • Controlled experiments: material presets, fill level, “air” presets
  • Multi-view capture and export (RGB, optional depth, instance masks)
  • Baseline evaluation (segmentation first; pose as optional extension)
